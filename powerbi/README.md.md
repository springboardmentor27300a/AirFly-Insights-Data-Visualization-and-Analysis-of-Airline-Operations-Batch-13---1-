📊 **Power BI Dashboards**


This folder contains the Power BI dashboards developed for the Flight Operations and Delay Analysis project.


📌 Overview



The dashboards provide a visual representation of flight operations, delays, cancellations, and performance metrics. They are designed to present insights in a clear and interactive manner.



📈 Dashboards Included



\- Flight Operations Dashboard

 Displays overall flight activity, airline performance, and route analysis



\- Delay Analysis Dashboard

 Highlights delay patterns, cancellation trends, and contributing factors
