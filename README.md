# AirFly Insights: Airline Data Analysis Dashboard

This project analyzes airline operations using Python and Power BI.

## Key Features:
- Flight delay analysis
- Cancellation insights
- Route and airport analysis
- Interactive dashboard

## Tools Used:
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Dataset:
- Airline dataset with ~5.8 million records
