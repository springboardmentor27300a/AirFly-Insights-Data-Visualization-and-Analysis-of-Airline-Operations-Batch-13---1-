# Internship Project – AirFly Insights

## 📌 Overview
This repository documents my internship project workflow on airline operations data.  
- **Phase 1 (Jupyter Notebook):** Data cleaning and preprocessing.  
- **Phase 2 (Google Colab):** Further analysis, visualization, and insights using the cleaned dataset.  
By separating these stages, the project demonstrates a complete pipeline: from raw dataset preparation to advanced analysis.
---
## 📂 Repository Structure
AirFly-Insights/
├── data_cleaning.ipynb       # Jupyter notebook with cleaning steps
├── cleaned_data.csv          # Cleaned dataset prepared in Jupyter
├── internship_colab.ipynb    # Colab notebook with further analysis
└── README.md                 # Documentation of workflow
---
## ⚙️ Workflow
1. **Data Cleaning (Jupyter Notebook)**  
   - Removed duplicates, handled missing values, normalized formats.  
   - Exported cleaned dataset as `cleaned_data.csv`.
2. **Analysis & Visualization (Colab Notebook)**  
   - Imported `cleaned_data.csv` into Colab.  
   - Performed exploratory data analysis (EDA).  
   - Built visualizations and documented findings.
