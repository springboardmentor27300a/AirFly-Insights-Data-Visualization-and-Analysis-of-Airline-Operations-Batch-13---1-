# ✈️ AirFly Insights – Flight Data Analytics Project

## 📌 About the Project

AirFly Insights focuses on analyzing airline flight data to understand operational performance, delay behavior, and route efficiency. The project uses data analysis and visualization techniques to transform raw flight records into clear and actionable insights.

---

## 🎯 Key Goals

* Examine how delays vary across airlines and routes
* Identify peak hours and time-based delay patterns
* Analyze seasonal trends in flight operations
* Understand major contributing factors to delays
* Present findings through an interactive dashboard

---

## ⚙️ Workflow

* Processed and cleaned the dataset to handle missing and inconsistent values
* Explored the data using statistical and visual analysis
* Built meaningful features such as routes and time-based attributes
* Developed a Power BI dashboard for better interpretation

---

## 🛠️ Tools Used

* Python (Pandas, NumPy)
* Visualization libraries (Matplotlib, Seaborn, Plotly)
* Power BI for dashboard creation
* Jupyter Notebook

---

## 📊 Highlights from Analysis

* Delay patterns are strongly influenced by departure time
* Certain routes consistently experience higher delays
* External factors like weather and aircraft availability play a key role
* Monthly trends indicate variations in flight performance

---

## 💡 Conclusion

The project demonstrates how data can be used to uncover hidden patterns in airline operations and support better decision-making through visualization and analysis.
