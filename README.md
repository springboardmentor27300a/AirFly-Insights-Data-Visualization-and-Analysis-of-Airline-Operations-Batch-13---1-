✈️ **AirFly Insights: Flight Operations \& Delay Analysis**



\---



📌 Objective



The objective of this project is to analyze airline operations, flight delays, and cancellations to understand performance patterns and identify operational inefficiencies.



\---



📊 Dataset



This project uses a dataset containing:



\- Airlines

\- Origin and destination airports

\- Departure and arrival delays

\- Flight cancellations

\- Time-based flight activity



A sample dataset is included in this repository for reference.



🔗 Full dataset (large size) is available here:

👉 https://drive.google.com/drive/folders/1W5-gU-8vJVuog7cnW_8YI76leV-Cn3qo?usp=drive_link



\---



🛠️ Tools Used



\- Power BI → Data visualization \& dashboard creation

\- Python (Pandas, NumPy) → Data preprocessing and feature engineering

\- CSV / Excel → Data source

\- GitHub → Project documentation and version control



\---



🔄 Project Workflow



1\. Data Understanding

  

  - Explored dataset structure and key attributes



2\. Data Cleaning \& Preprocessing

  

  - Handled missing values and inconsistencies

  - Standardized formats (date, time, categorical values)

  - Created new meaningful features for analysis



3\. Visualization Development

  

  - Built charts to analyze flight operations and delay patterns

  - Identified trends and relationships



4\. Dashboard Creation

  

        - Developed interactive dashboards in Power BI
        - Designed KPI cards and performance indicators



5\. Final Analysis \& Conclusion

&#x20;  

&#x20;  - Extracted insights and summarized findings



\---



📈 Dashboards



🔹 Flight Operations Dashboard



\- Displays overall flight activity and airline performance

\- Highlights busiest routes and airports

\- Shows departure patterns and monthly trends



🔹 Delay Analysis Dashboard



\- Analyzes delay patterns and cancellation trends

\- Identifies delay-prone routes and key factors

\- Evaluates overall operational performance
