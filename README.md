Project Overview -  
AirFly Insights is a data analysis project focused on understanding airline delays, cancellations, and performance trends.
The project uses Python for data processing and Power BI for dashboard visualization.

Objectives -
Analyze airline delay patterns,
Identify busiest airports and routes,
Study cancellation trends,
Build an interactive dashboard for insights.

Dataset -
Source: Airline Flights Dataset

Contains:
Airline information,
Departure & arrival delays,
Airports and routes,
Cancellation data,

Tools & Technologies Used -
Python :- Pandas , Matplotlib , Seaborn
Power BI (Dashboard creation),
Google Colab,
Microsoft Excel.

Project Workflow -
Data Cleaning (Milestone 1):
Handled missing values,
Optimized data types,
Created new features:
DATE,
MONTH_NAME,
DAY_NAME,
ROUTE,
DEP_HOUR.

Exploratory Data Analysis (Milestone 2):
Delay distribution analysis,
Airline performance comparison,
Monthly and hourly trends,
Visualization using charts.

Advanced Analysis (Milestone 3):
Airport-level delay analysis,
Route-level delay insights,
Cancellation trends,
Seasonal comparison (Winter vs Non-Winter).

Dashboard Creation (Milestone 4):
Created an interactive Power BI dashboard including:

🔹 KPI Metrics:-
Total Flights,
Average Delay,
Cancellation Rate,
Total Airlines.
🔹 Visualizations:-
Flights by Airline,
Delay by Airline,
Monthly Delay Trend,
Cancellation Trend,
Top Airports,
Route Delay Analysis,
Delay by Hour.
🔹 Filters:-
Airline,
Month,
Origin Airport,
Destination Airport.

Key Insights -
Some airlines experience higher delays than others,
Winter months show increased delays and cancellations,
Certain routes are more delay-prone,
Peak hours contribute to higher delays.

How to Run the Project -
Open the notebook in Google Colab,
Run all preprocessing steps,
Load cleaned dataset into Power BI,
Build dashboard using visuals.

Conclusion -
This project demonstrates how data analysis and visualization can help understand airline performance and delays, enabling better decision-making.
