# Project - AirFly Insights

## Project Overview
This is a data analysis and visualization project based on airline flight data. In this project, I 
analyzed a large scale flight dataset to understand delay patterns, cancellation reasons, airline 
performance, and airport trends. The purpose of this project is to transform raw flight data 
into meaningful insights using Python visualizations and Power BI dashboards.

## Projecct Goal
The goal of this project is to work on a large scale Airline flights dataset to analyze 
airline operations, flights distribution, delay reason, cancelation reason and operational trends
using data visualization that helps to improving airline and airport performance through actionable insights.

## Dataset
The dataset contains large-scale airline flight records .

### Key Features
- AIRLINE – Name of airline
- ORIGIN_AIRPORT – Departure airport
- DESTINATION_AIRPORT – Arrival airport
- DEPARTURE_DELAY – Delay in departure (minutes)
- ARRIVAL_DELAY – Delay in arrival (minutes)
- CANCELLED – Flight cancellation status
- CANCELLATION_REASON – Reason for cancellation
- DISTANCE – Flight distance
- DAY, MONTH, YEAR – Date details

## Tools Used
- Python
   (Pandas, NumPy) - Data Preprocessing, Feature Engineeing
   (Matplotlib, Seaborn, Plotly) - Data Visualization and Exploratory Data Analysis
- Power BI - Interactive reports and Dashboard
- Jupyter Notebook - coding
- GitHub - Project Documation and Version Control

## Project Workflow
1. Load dataset
2. Explore data structure and missing values
3. Perform data cleaning and preprocessing
4. Feature engineering for delays and cancellations
5. Conduct exploratory data analysis (EDA)
6. Create visualizations for trends and patterns
7. Build Power BI dashboard
8. Generate business insights and recommendations

## Exploratory Analysis Performed
In this project, I performed; Airline-wise delay analysis, Monthly delay trends, Top airports analysis,
Route-level delay comparison, Correlation heatmap, Distance vs delay analysis, Flight status distribution .
