# ✈️ AirFly Insights  
## Data Visualization and Analysis of Airline Operations

---

## 📌 Project Overview
**AirFly Insights** is a data analytics and visualization project focused on analyzing large-scale airline flight data to uncover patterns related to **delays, cancellations, routes, airports, and seasonal trends**.  
Using Python and Power BI, the project transforms raw aviation data into meaningful insights through structured preprocessing, visual exploration, and dashboard storytelling.

---

## 🎯 Project Objectives
- Analyze airline operational performance using real-world data  
- Identify delay patterns and cancellation reasons  
- Understand airport and route-level congestion  
- Discover seasonal and weather-related trends  
- Present insights using clear and interactive dashboards  

---

## 📂 Dataset Information
- **Source:** Kaggle – Airline Flights Dataset  - https://drive.google.com/drive/folders/1ETwTqMuErxtqweCep9I2fnj4DQmTJDWw?usp=sharing
- **Size:** 5+ million records  
- **Scope:** Domestic airline operations  
- **Key Attributes:**  
  Flight schedules, delays, cancellations, routes, airlines, airports, and delay causes  

---

## 🛠️ Tech Stack
- **Data Handling:** Python (pandas, numpy)  
- **Visualization:** Power BI, matplotlib, seaborn  
- **Dashboarding:** Power BI  
- **Documentation:** Jupyter Notebook, GitHub README

# 🚀 Project Milestones

---

## ✅ Milestone 1: Data Foundation and Cleaning

### 🎯 Objective
To prepare a clean, structured, and analysis-ready dataset by understanding the raw data and performing essential preprocessing steps.

### 🔧 What Was Done
- Loaded large CSV files using pandas  
- Explored dataset schema, column types, size, and missing values  
- Identified important analytical columns  
- Performed **sampling and memory optimization**  
- Handled missing values in delay and cancellation-related columns  
- Corrected data types for numeric and time fields  

### 🧩 Feature Engineering
- Created derived features:
  - Month  
  - Day of Week  
  - Hour of Departure  
  - Route (Origin–Destination)  
- Standardized datetime columns  

### ✅ Achievements
- Clean and optimized dataset  
- Feature dictionary created  
- Preprocessed data saved for reuse  

---

## ✅ Milestone 2: Visual Exploration and Delay Trends

### 🎯 Objective
To explore flight patterns and delay behaviors using univariate and bivariate visual analysis.

### 🔧 What Was Done
- Analyzed:
  - Top airlines by flight count  
  - Busiest routes and airports  
  - Flight distribution by month, day, and time  
- Created visualizations:
  - Bar charts  
  - Histograms  
  - Boxplots  
  - Line charts  
- Performed delay cause analysis:
  - Airline delays  
  - Weather delays  
  - NAS delays  

### ✅ Achievements
- Identified peak delay hours  
- Highlighted delay-prone airlines  
- Produced **8+ meaningful visualizations**  

---

## ✅ Milestone 3: Route, Cancellation, and Seasonal Insights

### 🎯 Objective
To analyze congestion, cancellations, and seasonal effects at airport and route levels.

### 🔧 What Was Done
- Identified **Top 10 origin–destination routes**  
- Conducted route-level delay analysis  
- Built **heatmaps**:
  - Airport vs Month  
  - Route vs Delay  
- Analyzed cancellation trends:
  - Monthly cancellation trends  
  - Cancellation reasons (Carrier, Weather, Security, NAS)  
- Compared **Winter vs Non-Winter cancellations**  

### ✅ Achievements
- Discovered highly congested routes  
- Identified winter months as peak cancellation periods  
- Found dominant cancellation causes  

---

## ✅ Milestone 4: Report and Dashboard Presentation

---

## 📊 Key Insights Summary
- Peak delays occur during specific hours and congested routes  
- Certain airlines and airports are consistently delay-prone  
- Winter months show higher cancellation rates  
- Weather is a major contributor to cancellations  
- Route congestion significantly affects operational efficiency
