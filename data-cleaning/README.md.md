🧹 **Data Cleaning \& Preprocessing**



This folder contains the data preprocessing steps performed on the flight dataset using Python.



📌 Overview



The raw dataset was processed to improve data quality and make it suitable for analysis and visualization. The cleaning process focused on ensuring consistency, accuracy, and usability of the data.



⚙️ Key Steps



\- Handled missing and null values

\- Removed duplicate and inconsistent records

\- Standardized data formats (date, time, categorical values)

\- Converted data types for proper analysis

\- Created new features such as delay metrics and derived columns



🎯 Purpose



The objective of this step was to transform raw data into a structured and meaningful format that can be used effectively for building dashboards and extracting insights.



🛠 Tools Used



\- Python

\- Pandas

\- NumPy



📊 Output



The cleaned dataset serves as the foundation for all visualizations and dashboards developed in this project.
