# Airline Delay Analysis Dashboard

## Project Description
This project analyzes airline delay performance using Microsoft Power BI.
The dashboard provides insights into delays, cancellations, and busiest routes
with interactive filters.

## Features
- KPI metrics (Total Flights, Avg Delay, Cancelled Flights)
- Delay by Airline
- Monthly Trend Analysis
- Cancellation Reasons
- Route Analysis
- Interactive Slicers

## Tool Used
- Microsoft Power BI
