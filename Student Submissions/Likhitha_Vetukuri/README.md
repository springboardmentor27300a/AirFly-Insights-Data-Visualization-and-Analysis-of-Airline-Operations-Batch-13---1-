# AirFly Insights – Airline Data Analysis

## Overview

This project focuses on analyzing airline operations data to uncover meaningful insights such as delays, cancellations, performance trends, and operational efficiency.

The analysis is performed using Python and data visualization techniques to help understand patterns in airline performance.

---

## Objectives

* Analyze flight delays and cancellations
* Identify patterns in airline performance
* Visualize trends using charts and graphs
* Provide insights for better decision-making

---

## Tools & Technologies

* Python 
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

## Key Analysis Performed

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Delay analysis (arrival & departure)
* Airline performance comparison
* Visualization of trends

---

## How to Run

1. Download the project files
2. Open `flights.ipynb` in Jupyter Notebook
3. Run all cells to view analysis and results

---

## Results

The project provides insights into:

* Which airlines perform best
* Common causes of delays
* Peak delay times
* Overall operational efficiency
